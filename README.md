# Portfolio

Portfolio site built with Go and HTMX.

## Stack

| Layer | Tech |
|---|---|
| Backend | Go |
| Interactivity | HTMX 2.x |
| Styling | Single `static/css/style.css`, no frameworks |

## Development

```bash
go run ./cmd/server/
```

## Configuration

| Env var | Default | Description |
|---|---|---|
| `PORT` | `8080` | HTTP listen port |
